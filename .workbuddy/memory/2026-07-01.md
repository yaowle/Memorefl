- **Persistence.kt**: 新增 `CardColorScheme` 数据类和 `cardColorPalette`（13 组背景+文字色配对）
- **Persistence.kt**: 新增 `getCardTextColor()` 工具函数，根据背景色返回配对文字色

- 所有配对在深色模式下对比度 ≥ 4.5:1（WCAG AA），确保文字清晰可读

## 项目熟悉

完成了对 memorefl 项目的全面代码审查，理解了项目全貌：

### 项目概况
- **名称**：记忆反射 (Memorefl)
- **类型**：Android 原生应用
- **技术栈**：Kotlin + Jetpack Compose + Room + MVVM
- **包名**：com.example.myapplication（开发阶段）
- **编译**：API 35 (Android 15)，最低 API 24 (Android 7.0)

### 核心架构
- **数据模型**：`KnowledgeNode` 递归树结构，序列化存储
- **存储**：Room 双表（`knowledge_schemes` + `nodes`），JSON 备份
- **架构**：MVVM (KnowledgeViewModel → KnowledgeRepository → Room DAO)
- **UI**：Material 3 Compose，深色主题

### 功能特性
1. 多方案管理（创建/重命名/删除/导入导出 JSON）
2. 权重瓦片布局（按权重自动分配屏幕空间）
3. 三种节点类型：分类(CATEGORY)、笔记(NOTE)、日历(CALENDAR)
4. 笔记支持富内容块（文本/图片/待办/文件）
5. 日历支持私有/全局日程，今日任务状态展示
6. 撤销/重做（20步历史）
7. 卡片颜色自定义（13种暗色配色 + 对比度配对）
8. 中/英文 i18n，应用内语言切换

### 项目结构
- `Persistence.kt`: 数据模型 + Room 数据库 + DAO
- `KnowledgeViewModel.kt`: 核心业务逻辑 + 状态管理
- `KnowledgeRepository.kt`: 数据仓库层
- `TreeLogic.kt`: 树操作工具类（查找/删除/移动/展平）
- `MainActivity.kt`: 入口 + 导航 + 三态切换（编辑/设置/浏览）
- `LanguageManager.kt`: 语言切换
- `ui/components/`: 6个 Compose 组件

### 遗留版本
- `Memorefl-master/`: 原始 XML View 版本，当前 Compose 是其重写版
