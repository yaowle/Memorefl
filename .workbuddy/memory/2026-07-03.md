# 2026-07-03 工作日志

## 项目熟悉 — MemoRefl Android 应用

### 项目概况
- 应用名: MemoRefl（记忆反射）
- 包名: com.example.myapplication
- 技术栈: Kotlin + Jetpack Compose + Room + kotlinx.serialization + Coil
- SDK: minSdk=24, targetSdk=35, compileSdk=35
- 构建工具: Gradle 8.3.2 + AGP, Kotlin 2.0.0, KSP

### 架构分层
- **UI层**: MainActivity(入口+导航), TreeEditor(树编辑器), WeightedTileLayout(权重磁贴), FunctionPage(功能页), NodeEditItem(节点编辑), SettingsPage(设置), NoteDetailEditor(便签编辑), CalendarEditorDialog(日历编辑)
- **业务层**: KnowledgeViewModel(状态管理+撤销重做+防抖保存), TreeLogic(树操作工具), LanguageManager(多语言zh/en)
- **数据层**: KnowledgeRepository -> AppDatabase(Room v4) -> KnowledgeScheme + NodeEntity

### 核心数据模型
- KnowledgeNode: id, title, weight, children, isDefault, limitDisabled, nodeType(CATEGORY/NOTE/CALENDAR), content(JSON), sharedCalendarEnabled, color
- 双存储策略: 关系型(nodes表) + JSON备份(jsonContent字段), 互为恢复依据
- NoteBlock: sealed class (Text/Image/Todo/File), 支持结构化便签

### 已完成的功能
1. 知识树导航 + 权重磁贴布局
2. 树编辑器(增删改移/折叠/层级缩进)
3. 便签编辑器(文本/标题/图片/文件/待办)
4. 日历功能(私有/全局共享) — 部分功能已隐藏
